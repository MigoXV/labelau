# LabelAU 使用说明

LabelAU 是一个用于 WAV 音频标注的桌面工具，支持浏览目录、播放音频、在波形和频谱上标注语音区间，并导出 Audition 兼容的 CSV 文件。

## 使用步骤

1. 点击“打开目录”，选择包含 WAV 文件的目录。
2. 系统会递归扫描目录，自动匹配同目录下同名的 CSV 标注文件。
3. 如果目录里存在损坏或不支持的 WAV，系统会自动跳过，并在界面底部提示跳过数量。
4. 从左侧文件列表选择一个音频，开始播放、查看波形和频谱。
5. 在波形图或频谱图中拖拽即可新增标注区间。
6. 切换到擦除工具后，再次拖拽可删除已有区间。
7. 点击“保存 CSV”后，当前标注会写入与音频同名的 CSV 文件。

## 常用操作

- 左侧列表可按文件名、目录和任务状态筛选。
- “未处理 / 未保存 / 已处理”可快速查看任务进度。
- 波形图适合快速浏览时间范围，频谱图适合精细调整。
- 多声道音频可在频谱上方切换声道查看。
- 如果当前文件有未保存修改，关闭应用时会提示是否保存后退出。

## 快捷键

- `空格`：播放 / 暂停
- `S` 或 `Ctrl+S` / `Cmd+S`：保存当前 CSV
- `Ctrl+Z` / `Cmd+Z`：撤销上一步标注
- `M`：切换到标注工具
- `E`：切换到擦除工具
- `V`：切换到拖拽查看模式
- `X`：切换线性 / 对数频率缩放
- `R`：切换播放倍率
- `↑ / ↓`：切换上一条 / 下一条音频
- `← / →`：平移当前时间窗口

## 注意事项

- 当前仅支持 WAV 音频文件。
- CSV 会保存为与音频同目录、同文件名的 `.csv` 文件。
- 未保存状态会在界面底部显示，请在切换任务或退出前确认是否需要保存。
- 如果目录中没有可用 WAV，界面会提示未找到可导入文件。
